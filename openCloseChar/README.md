# ClosingCharTester
is a small program made in C and Python (choose the one you like) to test if the openning char ( [ { are closed respecting the order.  
it uses a linked list to represent a stack.

# Example
"(([{}]))" 		return TRUE  
"(([{})])" 		return FALSE  
"()([{}])()" 	return TRUE  
"([{]})" 		return FALSE
